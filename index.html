D&D 5e Point-Buy Calculator

About

This point-buy calculator is a work in progress and aims to cover the following D&D 5e books:

  • Player's Handbook
  • Dungeon Master's Guide
  • Unearthed Arcana: Eberron
  • Unearthed Arcana: Gothic Heroes
  • Unearthed Arcana: Waterborne Adventures
  • Elemental Evil Player's Guide

It includes the ability score point cost for each increment between 8 and 15, racial bonuses and penalties based on race and subrace selection, a total ability score, the resulting ability score modifier, a total point cost, and a summary of all racial traits at the end. There's also a reset button following the racial traits section.

Basic Usage

Recommendations
Use 27 points, as recommended by the Player's Handbook.
Variants
If there is an option to select a variant (Subrace), you must pick one (with the exception of Humans). The Races which do not have this option, do not require one to be selected.
